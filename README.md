# NLP Projects

This repository contains Natural Language Processing (NLP) projects built using Movie Reviews and Twitter datasets.

## Datasets

* Movie Reviews Dataset
* Twitter Dataset

## NLP Techniques

* Bag of Words (BoW)
* TF-IDF
* Word2Vec

## Tasks Performed

* Text Preprocessing
* Tokenization
* Stopword Removal
* Lemmatization
* Feature Extraction
* Text Classification
* Sentiment Analysis

## Tools & Libraries

* Python
* Pandas
* NumPy
* NLTK
* Gensim
* Scikit-Learn
* Jupyter Notebook
